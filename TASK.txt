Задача: Дается 10000 ip-адресов серверов. 
Необходимо опрашивать адреса на доступность (up/down). 
Как только сервер стал недоступен выдавать в консоль dd-MM-yyyy hh:mm:ss.fff, ip:port down, если сервер поднялся выдавать dd-MM-yyyy hh:mm:ss.fff, ip:port up. 
Список серверов хранится в файле в виде ip:port. 
Вывод в консоль и файл. 
Для работы с сетью использовать только системные вызовы (для управления потоками можно использовать библиотеку pthread).

Критерий недоступности -- не получилось установить tcp-соединение. Попытка одна, время ожидания установления соединения стандартный таймаут tcp для установления соединения.
Проверяется только факт соединения по сокету. Реалировать изменение необходимо как можно быстрее, насколько позволяют физические ограничения хоста. 
После завершения полного цикла опроса всех 10к адресов можно сделать паузу в  1сек.
